Runing the script, after installing theano:
-python ./LIG-Doctor.py ./data/mimic-90-10_01_271 ./data/mimic-90-10_01_271.DURATION ./output

The file names "./data/mimic-90-10_01_271" and "./data/mimic-90-10_01_271.DURATION" are only radical to the actual names, the script will look for files "./data/mimic-90-10_01_271.test", "./data/mimic-90-10_01_271.train", "./data/mimic-90-10_01_271.DURATION.train", and "./data/mimic-90-10_01_271.DURATION.test".
